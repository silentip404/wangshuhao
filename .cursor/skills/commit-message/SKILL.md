# Git Commit Message Generation Standards

This skill applies when generating Git commit messages.

---

## Required Reference

When generating Git commit messages, you **MUST** read and strictly follow the specifications in [.cursor/commands/commit-message.md](.cursor/commands/commit-message.md). You **MUST NOT** reference any other commit message configurations, conventions, or formats (e.g., Angular commit guidelines, or repository commit history).
