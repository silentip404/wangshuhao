# ESLint 配置目录说明

本目录包含项目中的 ESLint 配置文件，用于统一管理代码规范和检查规则。

## 📁 目录结构

```
eslint-configs/
├── README.md          # 本说明文档
├── overrides/         # 覆盖规则目录（按规则来源分类）
└── presets/           # 预设配置目录
```

## 📝 目录说明

### `overrides/`

项目自身维护的覆盖规则目录，按规则来源进行分类管理。每个文件对应一类规则来源的配置，这些配置在 presets 基础上进行补充和覆盖，便于维护和复用。

### `presets/`

社区维护的预设配置目录，包含针对不同场景的 ESLint 配置预设。
